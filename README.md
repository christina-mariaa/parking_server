# EasyPark Mobile App

Серверная часть автоматизированной системы управления автостоянкой.

Отвечает за бизнес-логику, хранение данных, бронирование мест,
аутентификацию пользователей и управление доступом через QR-коды.

Все клиенты системы (мобильное приложение, десктоп-приложение администратора,
контроллер въезда) взаимодействуют с сервером через REST API и WebSocket.
